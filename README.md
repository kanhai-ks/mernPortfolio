 MERN Portfolio 

A responsive developer portfolio built with the MERN stack using Express, MongoDB, React, Node and TailwindCSS for style.  
This project showcases my skills, projects and professional experience with a modern and mobile-first design.


  Features
- Responsive design with TailwindCSS
- Full-stack setup using MERN
- SEO optimized with Meta Tag
- Project showcase section with live demos and GitHub links
- Contact form with backend API integration
- Deployed on Vercel with DNS configuration

  Getting Started
 
 Clone the repository
 git clone https://github.com/kanhai-ks/mernPortfolio.git
 cd mernPortfolio
